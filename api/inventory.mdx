## Overview

The Inventory API provides comprehensive endpoints for managing inventory items, including CRUD operations, quantity updates, and analytics. All endpoints require authentication.

## Get All Inventory

<api method="GET" endpoint="/api/inventory" />

Retrieve all inventory items with pagination and filtering.

**Access:** Private (all authenticated users)

### Query Parameters

<ParamField query="page" type="number" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="number" default="10">
  Number of items per page
</ParamField>

<ParamField query="category" type="string">
  Filter by category (case-insensitive)
</ParamField>

<ParamField query="supplier" type="string">
  Filter by supplier (case-insensitive)
</ParamField>

<ParamField query="search" type="string">
  Search by product name or SKU
</ParamField>

<ParamField query="minQty" type="number">
  Minimum quantity filter
</ParamField>

<ParamField query="maxQty" type="number">
  Maximum quantity filter
</ParamField>

<ParamField query="sortBy" type="string">
  Field to sort by (e.g., quantity, expiryDate)
</ParamField>

<ParamField query="order" type="string">
  Sort order: `asc` or `desc`
</ParamField>

### Response

<ResponseField name="items" type="array">
  Array of inventory items
</ResponseField>

<ResponseField name="pagination" type="object">
  <ResponseField name="currentPage" type="number">
    Current page number
  </ResponseField>
  <ResponseField name="totalPages" type="number">
    Total number of pages
  </ResponseField>
  <ResponseField name="totalItems" type="number">
    Total number of items
  </ResponseField>
  <ResponseField name="limit" type="number">
    Items per page
  </ResponseField>
</ResponseField>

### Example Response

{
  "success": true,
  "message": "Inventory items fetched successfully",
  "data": {
    "items": [
      {
        "_id": "65a1234567890abcdef12345",

        "quantity": 150,

## Create Inventory Item

<api method="POST" endpoint="/api/inventory" />

Create a new inventory item.

**Access:** Private (Admin/Manager only)

### Request Body

<ParamField path="productName" type="string" required>
  Name of the product
</ParamField>

<ParamField path="category" type="string" required>
  Product category
</ParamField>

<ParamField path="sku" type="string" required>
  Stock Keeping Unit (will be converted to uppercase)
</ParamField>

<ParamField path="quantity" type="number" required>
  Initial quantity in stock
</ParamField>

<ParamField path="purchaseDate" type="string" required>
  Purchase date (ISO 8601 format)
</ParamField>

<ParamField path="expiryDate" type="string" required>
  Expiry date (must be after purchase date)
</ParamField>

<ParamField path="supplier" type="string" required>
  Supplier name
</ParamField>

### Example Request

```json
{
  "productName": "Organic Apples",
  "category": "Fruits",
  "sku": "FRT-001",
  "quantity": 150,
  "purchaseDate": "2024-01-15",
  "expiryDate": "2024-02-15",
  "supplier": "Fresh Farms Ltd"
}
```

## Update Quantity

<api method="PATCH" endpoint="/api/inventory/:id/quantity" />

Quickly update inventory quantity with add, subtract, or set operations.

**Access:** Private (Admin/Manager only)

### Request Body

<ParamField path="quantity" type="number" required>
  Quantity value
</ParamField>

<ParamField path="operation" type="string" required>
  Operation type: `add`, `subtract`, or `set`
</ParamField>

### Example Request

```json
{
  "quantity": 50,
  "operation": "add"
}
```

### Example Response

```json
{
  "success": true,
  "message": "Inventory quantity updated successfully",
  "data": {
    "item": {
      "_id": "65a1234567890abcdef12345",
      "productName": "Organic Apples",
      "quantity": 200,
      "lastModifiedBy": {
        "name": "Manager User",
        "email": "manager@example.com"
      }
    }
  }
}
```
